# Qt: 网易云音乐
一款基于 Qt5 的网易云音乐播放器，已编译好的 NeteaseMusic 适用于 Linux 64 位 Qt 5 环境。  
已完成功能：排行榜点播，搜索歌曲，MV。  
音乐版权归网易所有。
